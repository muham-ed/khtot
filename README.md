# Arrow Puzzle (khtot) 🏹

**Arrow Puzzle** هي لعبة ألغاز منطقية ممتعة ومريحة للأعصاب مصممة خصيصاً لمنصة أندرويد. الهدف من اللعبة هو إخراج جميع الأسهم من المتاهة بالترتيب الصحيح.

---

## 🌟 مميزات اللعبة (Features)

*   **مستويات لا نهائية:** نظام توليد مستويات عشوائي (Procedural Generation) يزداد صعوبة كلما تقدمت.
*   **أداء عالٍ:** تم تطوير اللعبة باستخدام **C++** ومكتبة **Android Game Activity** لضمان سلاسة تامة.
*   **تصميم مريح:** ألوان هادئة وواجهة بسيطة تساعد على التركيز والاسترخاء.
*   **تحدي ذكاء:** تتطلب اللعبة تفكيراً مكانياً ومنطقياً لتحديد المسارات المفتوحة.
*   **نظام محاولات:** حافظ على تركيزك، فكل حركة خاطئة تكلفك محاولة!

---

## 🛠 التقنيات المستخدمة (Tech Stack)

*   **اللغات:** C++ (Core Logic), Kotlin (Android Wrapper).
*   **الرسوميات:** OpenGL ES 3.0.
*   **المكتبات:** 
    *   `GameActivity` (Android Game SDK).
    *   `CMake` لبناء الكود المصدري.
    *   `GLM` (أو حسابات رياضية مخصصة للتحويلات).

---

##  كيفية التشغيل (Getting Started)

1.  قم بتحميل المشروع باستخدام Git:
    ```bash
    git clone https://github.com/mohamedalaa/khtot.git
    ```
2.  افتح المشروع باستخدام **Android Studio** (نسخة Ladybug أو أحدث).
3.  تأكد من تثبيت **NDK** و **CMake** من خلال SDK Manager.
4.  قم بعمل **Sync Project with Gradle Files**.
5.  اضغط على **Run** لتشغيل اللعبة على محاكي أو جهاز حقيقي.

---

##  طريقة اللعب (How to Play)

*   اضغط على السهم لتحريكه.
*   ينطلق السهم فقط إذا كان المسار أمامه خالياً تماماً من الأسهم الأخرى.
*   تنتهي المرحلة عند إخلاء الشاشة بالكامل.
*   إذا ضغطت على سهم مسدوده طريقه، ستخسر محاولة.
